# Weather Dashboard

A full-stack weather dashboard application built with React, Node.js, Express, and MongoDB.

## Features
- Real-time weather data from OpenWeatherMap API
- 5-day weather forecast
- User authentication (Registration/Login)
- Favorite cities management
- Geolocation support
- Unit preference (Metric/Imperial)

## Setup
1. Clone the repository.
2. Run `npm run install-all` in the root directory.
3. Configure your `.env` file in the `server/` directory.
4. Run `npm start` to start the application.
